# OpenCV-Notes in Python 3

### A collection of the principals algorithms used in OpenCV.

## Basic Usage
  * Reading Images and Videos
  * Resizing and Rescaling Images/Frames
  * Principal Functions
  * Shapes and Text
  * Transformations on Images
  * Contour Detection

## Advance Usage
  * Color Spaces
  * Color Channels
  * Blurring
  * Bitwise Operations
  * Masking
  * Image Histogram
  * Thresholding
  * Edge Detection
  * Face Detection
